6. FAQ
Como apago tudo?

Aba DADOS → Apagar todos os dados.

Cargo sem ninguém?

É "vaga em aberto". Útil pra planejar contratação.

Como excluir?

Clique no card → botão vermelho Excluir.

Dados somem?

Não. O app preserva o que está no seu navegador.

Duas empresas?

Exporte JSON de cada uma e alterne importando.

Sincroniza entre dispositivos?

Não. Use Exportar/Importar JSON pra mover.

Funciona offline?

Sim, depois da primeira visita.

Mudar cargo de alguém?

Clique na pessoa → troque no campo "Cargo".
